# Carcatalog
Car Catalog using ReactJS, Axios

## Prerequisites
- a running local MongoDB
- npm installed
- yarn installed

## Installation
`cd carcatalog-api
yarn install
`

`cd carcatalog-react
yarn install
`
## Running the application
`cd carcatalog-api
yarn start`

`cd carcatalog-react
yarn start`

browser should open
